워크플로우 제출하기

January 27, 2025 · 1 min · QuantNode

🚀 n8n 워크플로우 공유하기#

커뮤니티와 여러분의 워크플로우를 공유해주세요!

📋 제출 방법#

Option 1: GitHub Issue로 제출 (추천)#

  1. 워크플로우 제출 Issue 생성
  2. 템플릿에 따라 정보 입력
  3. n8n JSON 파일 첨부
  4. 스크린샷 첨부 (선택사항)

Option 2: 이메일로 제출#

📧 contact@quantnode.kr로 다음 정보와 함께 전송:

  • 워크플로우 제목
  • 설명 (용도, 주요 기능)
  • 카테고리 (자동매매, 분석, 알림 등)
  • n8n JSON 파일
  • 스크린샷 (선택사항)

📝 제출 가이드라인#

필수 정보#

  • 제목: 명확하고 설명적인 제목
  • 설명: 워크플로우의 목적과 주요 기능
  • 카테고리:
    • 🔄 자동매매
    • 📊 데이터 분석
    • 📢 알림/모니터링
    • 🛠️ 유틸리티
  • 난이도: 초급 / 중급 / 고급
  • JSON 파일: n8n에서 Export한 워크플로우 파일

선택 정보#

  • 스크린샷 또는 다이어그램
  • 설정 가이드
  • 필요한 API 키나 설정
  • 사용 팁

🎯 워크플로우 예시#

📈 업비트 자동매매 워크플로우#

{
  "name": "Upbit Auto Trading",
  "nodes": [
    {
      "name": "Schedule Trigger",
      "type": "@n8n/n8n-nodes-base.scheduleTrigger"
    },
    {
      "name": "HTTP Request - Get Price",
      "type": "@n8n/n8n-nodes-base.httpRequest"
    }
  ]
}

✅ 검토 과정#

  1. 제출: GitHub Issue 또는 이메일
  2. 검토: 관리자가 워크플로우 검토 (1-3일)
  3. 승인: 사이트에 추가 및 자동 배포
  4. 알림: 제출자에게 승인 완료 알림
