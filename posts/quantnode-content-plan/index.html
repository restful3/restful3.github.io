1. 사이트 개요 및 비전#

  • 사이트명: QuantNode.kr
  • 슬로건: “쉽게 배우고 자동화하는 퀀트 투자”
  • 핵심 가치: No-Code / 프롬프트 기반 퀀트 자동화, 실용 중심 튜토리얼, 커뮤니티 기반 성장
  • 타겟층: 퀀트 투자에 관심 있는 일반 투자자 (개인 투자자, 비개발자 포함)
  • 중장기 목표:

    • 책, 강의, 웨비나 등 교육 콘텐츠 판매

3. 콘텐츠 포맷 및 스타일 가이드#

  • 톤: 친절하고 대화형, 가벼운 유머 허용, 비전문가도 이해 가능한 수준
  • 형식: 마크다운 기반 블로그 (Hugo, PaperMod 기반), 시각자료 다수 포함
  • 기본 구성:
    • 도입 (왜 이 내용을 다루는가?)
    • 본문 (스크린샷, 실습, 프롬프트, 코드 등)
    • 정리 (Key Takeaways)
    • 실습 링크 또는 추가 참고 자료
  • 예외 사례: 인터뷰는 Q&A 스타일, 전략 리뷰는 스토리 기반

4. 콘텐츠 운영 계획 (출시 후 4주간)#

주차 콘텐츠 유형 제목 예시
1주차 Basics + Tutorial “퀀트란 무엇인가요?” / “n8n으로 전략 만들어보기”
2주차 Prompt Lab + Market “ChatGPT로 기업분석 해보기” / “이번 주 시장을 5차트로 본다면”
3주차 Strategy + Community “마법공식 전략 2025 리포트” / “유저 인터뷰: 처음 써본 자동화”
4주차 How-To + Event 예고 “Google Sheets 자동화 워크플로우” / “4월 웨비나 미리보기”
  • 이후 월별 릴리즈 캘린더 운영: 월간 전략 리포트 / 주간 마켓 브리핑 / 격주 튜토리얼

5. 커뮤니티 성장 전략#

단계별 전환 흐름#

  1. 유입: 블로그, 검색, 유튜브 쇼츠, 인스타 카드뉴스
  2. 가입: 뉴스레터 유도 / 워크플로우 템플릿 다운로드 조건
  3. 활성화: 댓글, AMA 참여, 슬랙/디스코드 초대
  4. 전환: 강의/책/웨비나 판매 연결 (Soft CTA)

주요 자산#

  • Starter Kit: QuantNode 입문자용 ZIP (PDF 튜토리얼 + 워크플로우 JSON)
  • 워크플로우 마켓: 유저가 공유하는 n8n 전략 흐름 예제 모음
  • Monthly Challenge: “이 전략을 개선해보세요!” 식 커뮤니티 참여형 콘텐츠

6. 향후 확장 로드맵#

단계 기능 또는 제품 목적
1단계 콘텐츠 플랫폼 (기초, 튜토리얼) 트래픽 확보, SEO 기반 유입
2단계 뉴스레터/세미나/웨비나 운영 커뮤니티 강화 및 데이터 수집
3단계 전자책/강의 출시 수익화 시작
4단계 QuantNode 솔루션 (MCP 기반 SaaS) 실전 자동화 투자 도구 유료화
5단계 B2B 제휴 및 기관향 자동화 API 판매 수익 다변화 및 신뢰도 제고
